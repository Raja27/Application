# Application

## Installation
> Pre-requirement Python 3.5
```
pip install -r requirement.txt
```

## Run server
```
./manage.py runserver
```
URL: 127.0.0.1:8000

## Software Used
1. Python 3.5
  - Django 1.11
  - Django rest framework 3.7
  - Django multi token
2. Angularjs 1.6
3. Bootstrap 3.3
